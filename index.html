Simulador de Impacto Inflacionario

Calculadora económica para medir cómo pierde valor el dinero en base a una tasa estimada.

# Simulación económica
precio_inicial = float(input("Precio actual del producto ($): "))
inflacion = float(input("Porcentaje de inflación mensual (%): "))

precio_final = precio_inicial * (1 + (inflacion / 100))
print(f"El mes que viene costará: ${precio_final:.2f}")
